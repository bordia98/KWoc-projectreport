KWOC Project Report



About KWOC

KWOC is a program to promote open source development among young generation. A wonderful initiative initiated by the IIT-KGP students to involved the students all over India to contribute towards open source development and prepare the students for the Google Summer of Code.

Choosing the project

Choosing a project in KWOC is one of the difficult task as there are a lot of choices available and you have to search them and understand to what extend you can contribute to it. As when I started KWOC I had started working on Django framework so for me the best thing was to search for a Django project to which I can contribute. So I choosed the project ACAD-Search which was under this domain selected an issue of it and started working on it.
